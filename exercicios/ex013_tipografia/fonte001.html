Famílias das fontes

Fontes serifadas

Fontes serifadas são mais usadas em impressões, poís é mais fácil a leitura no papel.

Fontes não serifadas

Fontes não serifadas são mais comuns na web pois, é mais fácil a leitura em tela.

Fontes Monoespaçadas

Nas fontes Monoespaçadas,a largura das letras são iguais. Esta fonte é usada em scripts

Fontes Script ou Handwriting

Fontes Script ou Handwriting são feitas para parecer que foram escritas a mão

Fontes Display

Fontes Display são usadas em logos ou títulos apenas, pois sua leitura é cansativa.
